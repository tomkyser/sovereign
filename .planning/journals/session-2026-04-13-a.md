## Key Findings

### Installer Scripts Are Just Downloaders
- `install.sh` and `install.ps1` are thin downloaders — they fetch the binary, verify SHA256 from `manifest.json`, run `claude install`, delete temp file
- The binary's own `claude install` command handles directory setup
- Real paths come from XDG Base Directory spec in `cc-source/src/utils/xdg.ts`:
  - Versions: `$XDG_DATA_HOME/claude/versions` → `~/.local/share/claude/versions`
  - Executable: `$HOME/.local/bin/claude`
- GCS has `manifest.json` per version with per-platform SHA256 checksums
- GCS has `latest` file returning current version string
- Platform strings: `darwin-arm64`, `darwin-x64`, `linux-x64`, `linux-arm64`, `linux-x64-musl`, `linux-arm64-musl`, `win32-x64`, `win32-arm64`
