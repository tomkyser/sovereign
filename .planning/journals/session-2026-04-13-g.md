### Key Discovery

CC's native GlobTool uses `--no-ignore --hidden` by default (glob.ts lines 98-99) because it serves the permission system. Our REPL glob intentionally deviates — scripting context expects .gitignore respected. Both `--no-ignore` AND `--hidden` together caused the 10K file explosion (pnpm stores packages under hidden `.pnpm` directories).
