## Key Technical Lessons

1. esbuild preserves readable structure — \s* between tokens for both formats
2. Variable names differ from Bun minifier — discover from context, not hardcode
3. Bun moduleFormat=0 (CJS) entry modules do not auto-execute
4. Pieces system splits at ${} boundaries — include ${ and } in surrounding text
5. Bundled data should take precedence over external downloads
