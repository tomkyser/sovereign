### Bugs Found and Fixed

**Bug 1: Replace mode breaks REPL delegation (F13)**
- Replace mode filters tools from the array that REPL delegates to
- `findTool('Read')` returns null because Read was filtered from context.options.tools
- Fix: Binary-patched loader stashes filtered tools on REPL tool object (`_stashedTools`)
- `findTool()` checks stash first, then context
- Commit: `0358f47`

**Bug 2: summarizeResult crash on undefined**
- `JSON.stringify(undefined).substring()` throws
- Read handler returned undefined for notebook results
- Fix: null guard in summarizeResult, defensive Read fallback for non-text results
- Commit: `0358f47`

**Bug 3: VM realm SyntaxError (F14)**
- `err instanceof SyntaxError` fails across VM realms (different constructors)
- IIFE fallback never triggered for await scripts
- Fix: String-based `err.name === 'SyntaxError'` check
- Fixed by CC model during test (auto-applied via hooks)
- Commit: `512cab7`

**Bug 4: IIFE wrapping kills variable persistence (F15)**
- `(async () => { const x = 42 })()` — x is function-scoped, lost
- Fix: Two-pass execution (sync first for var persistence, IIFE fallback for await/return)
- Added `state` object for explicit cross-call persistence
- Commit: `512cab7`

**Bug 5: `return` in sync scripts is SyntaxError**
- `return` is only valid inside a function body
- Sync pass runs script as raw eval where `return` is invalid
- Fix: Broadened IIFE fallback to trigger on any SyntaxError (not just await)
- Commit: `a6461d6`
